simulation module¶

simulation.create_agents(env, motif_length)[source]¶

Create a number of agents for the simulation. If you want to try out different things, modify this.

Parameters

  • env (MusicEnvironment) – The environment for the agents.

  • motif_length (float) – The length of the motifs that the agents use in quarter notes.

simulation.run_simulation(num_steps)[source]¶

Runs the simulation.

Parameters

num_steps (int) – Number of steps to run in simulation.
