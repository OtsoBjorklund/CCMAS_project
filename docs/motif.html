get_string_representation()[source]¶

Create a transposition invariant string representation of motif. The representation ignores octaves and treats durations relatively. Each pair of consecutive notes in the string is represented by the interval in half-steps and the ration of the durations. For example the pair of notes C4(1/8) - D4(1/4) is represented by 22.00 where the interval is 2 and the ratio of durations is 2.00 as the second note is twice as long as the first one. The ratio is limited to 2 decimal places to avoid the ratio part being extremely long. The first note of the motif is compared to the last note of the motif.

Returns

String of interval and duration numbers.

Return type

str

static transpose_to_key(motif, original_key, target_key)[source]¶

Transpose motif from original_key to target_key. Computation is performed using the number of sharps in the key signature. If the key signature contains 2 sharps for example, the tonic of that key is 2 perfect fifths higher than the pitch C. This transposition can be reduced to within an octave by computing the required transposition in half-steps and taking mod 12.

Parameters

  • motif (Motif) – Motif to be transposed.

  • original_key (music21.key) – The original key of the motif.

  • target_key (music21.key) – The target key for the motif.
