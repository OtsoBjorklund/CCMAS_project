musicxmlio module¶

musicxmlio.get_random_filename(path)[source]¶

Get the relative path of a random file from directory specified by path. :param path: The path of the directory from which to select random file. :type path: str :return: Path to randomly selected file. :rtype: str

musicxmlio.parts_to_musicxml_midi(parts, filename)[source]¶

Write the parts dictionary into a MusicXML file and MIDI file with name filename. Each part in the parts will be its own part in the MusicXML score and its own channel in MIDI file.

Parameters

  • parts (Dictionary with strings as keys and lists of Motif objects as values. The keys will be used as part names in the score.) – The parts to be written in to the score.

  • filename (String.) – filename for the MusicXML file.

musicxmlio.read_musicxml_to_list(filepath)[source]¶

Read all the parts in the MusicXML file and return a list of note or rest objects. Ignores everything other than Note and Rest objects. The parts are concatenated to each other so that parts that were simultaneous in the score are sequential in the list.

Parameters

filepath (String.) – The path to the MusicXML file to be read.

Returns

Returns a list of notes and rests.

Return type

List of music21.note.Note and music21.note.Rest objects.

musicxmlio.select_random_motifs(filename, motif_length, num_motifs)[source]¶

Select random sequences of notes as motifs from a MusicXML file.

Parameters

  • filename (String.) – Name of the file from which motifs will be selected.

  • motif_length (Float.) – The length of the motifs to be selected in quarter notes.

  • num_motifs (Integer.) – Number of motifs to be selected from file.

Returns

List of num_motifs motifs with length motif_length.

Return type

List of motif.Motif objects.
